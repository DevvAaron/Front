# Sistema de Gestión de Almacén (WMS) - Interfaz Front-End 🖥️📦

Esta es la interfaz de usuario moderna y responsiva desarrollada en **React** para el Sistema de Gestión de Almacén de **Alarmas Cobra SAC**. Diseñada para optimizar la operación y visualización de inventarios.

## 🔗 [VER CÓDIGO DEL REPOSITORIO](https://github.com/DevvAaron/Front)

*(Este proyecto se conecta a la API Backend alojada en [este repositorio](https://github.com/DevvAaron/Backend))*

---

### 🚀 Desafíos Técnicos Resueltos
Como parte del sistema para Alarmas Cobra, este Front-End aborda:

1. **Navegación Intuitiva (SPA):** Implementación de `react-router-dom` para una experiencia de usuario fluida sin recargas de página.
2. **Consumo de API REST:** Integración completa con el backend para visualización, creación, actualización y eliminación de activos en tiempo real.

---

### 🛠️ Stack Tecnológico
* **Core:** React 18+.
* **Navegación:** React Router Dom.
* **Estilizado:** CSS3 Moderno (Flexbox/Grid) con enfoque Mobile-First.
* **Herramientas:** Vite (para desarrollo rápido).
